# TranslateKB

A custom iOS keyboard that translates your draft in place with one tap.
Type → tap → your message becomes English (or any of 16 languages) — without
leaving WhatsApp, Telegram, Mail, or any other app.

Backed by Apple's on-device Translation framework. No accounts, no backend,
no analytics. Free.
